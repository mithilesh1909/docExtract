How It Works
Three steps from upload to structured data
01
Upload Document
Drop any PDF, DOCX, TXT, or image file. OCR handles scanned documents automatically.
02
AI Detection
GPT-4o-mini reads the content and classifies the document type — invoice, legal, resume, medical, or general.
03
Field Extraction
GPT-4o uses a type-specific prompt to extract every named field into a clean JSON structure.
04
Export & Integrate
Copy the JSON or hit the REST API directly at /api/extract from any app.
